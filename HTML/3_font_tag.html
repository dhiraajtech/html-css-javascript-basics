Text Formatting Tags

<font> Tag with Additional Attributes

Use the <font> tag to define font-related styles with additional attributes.

Use the font tag with the size="4" attribute to set the font size to 4.

Use the font tag with the face="Arial, sans-serif" attribute to set the font family to Arial, falling back to a sans-serif font if Arial is not available.

For more details, refer to the MDN HTML documentation.
