Basic HTML Concepts

<!DOCTYPE html>

The <!DOCTYPE html> declaration defines the document type and version of HTML being used. It must be placed at the very beginning of an HTML document.

            <!DOCTYPE html>

HTML Tag Attributes

HTML tags can have attributes that provide additional information about the element. Attributes are added within the opening tag.

            <tagname attribute="value">Content</tagname>

Example: <a href="https://example.com" target="_blank" title="Visit Example">Visit Example</a>

<head> Tag

The <head> tag contains meta-information about the HTML document, such as title, styles, and meta tags.

            <head>
    <title>Page Title</title>
</head>

<style> Tag

The <style> tag is used to define styles for the document, either inline or by referencing an external stylesheet.

            <style>
    body {
        background-color: #f0f0f0;
    }
</style>

<meta> Tag

The <meta> tag provides metadata about the HTML document, such as character set, viewport settings, etc.

            <meta charset="UTF-8">

<meta> Tag

The <meta> tag provides metadata about the HTML document.

Charset

            <meta charset="UTF-8">

Description, Author, Keywords

            <meta name="description" content="A sample HTML document covering basic HTML concepts.">
            <meta name="author" content="Your Name">
            <meta name="keywords" content="HTML, CSS, JavaScript, Web Development">

Viewport Settings

            <meta name="viewport" content="width=device-width, initial-scale=1.0">

<body> Tag

The <body> tag contains the content of the HTML document, including text, images, links, etc.

            <body>
    <h1>Hello, World!</h1>
    <p>This is a paragraph.</p>
</body>

Heading Tags (<h1> to <h6>)

Heading tags are used to define headings. <h1> is the largest, and <h6> is the smallest.

            <h1>Main Heading</h1>

<p> Tag

The <p> tag defines a paragraph of text.

            <p>This is a paragraph of text.</p>

<a> Tag

The <a> tag defines a hyperlink, linking to another page or resource.

            <a href="https://example.com" target="_blank" title="Visit Example">Visit Example</a>

<img> Tag

The <img> tag is used to embed images in the document.

            <img src="image.jpg" alt="Description">

<div> Tag

The <div> tag is a container that groups other HTML elements and applies styles or scripts to them.

            <div class="container">
    <p>This is a content inside a div.</p>
</div>

For more details, refer to the MDN HTML documentation.
