HTML Tags Showcase

<table> Tag

<table> <tr> <th>Header 1</th> <th>Header 2</th> </tr> <tr> <td>Row 1, Cell 1</td> <td>Row 1, Cell 2</td> </tr> </table>

Sample table:

Header 1 Header 2
Row 1, Cell 1 Row 1, Cell 2

The <table> tag is used to create an HTML table, and it can contain various elements such as <tr> (table row), <th> (table header), and <td> (table data).

<table> Tag with Colspan

<table> <tr> <th colspan="2">Header Row with Colspan</th> </tr> <tr> <td>Row 1, Cell 1</td> <td>Row 1, Cell 2</td> </tr> <tr> <td colspan="2">Row 2, Cell 1 and Cell 2 with Colspan</td> </tr> </table>

Sample table 2:

Header Row with Colspan
Row 1, Cell 1 Row 1, Cell 2
Row 2, Cell 1 and Cell 2 with Colspan

The colspan attribute is used to define the number of columns a table cell should span.

For more details, refer to the MDN HTML documentation.
